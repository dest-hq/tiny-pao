# tiny_pao

A simple CPU renderer for Rust.

**Work in Progress** - Currently just renders solid colors

## Features
- [x] Window integration via raw-window-handle
- [x] Clear screen with color
- [x] Resize support
- [ ] Draw rectangles
- [ ] Draw circles
- [ ] Anti-aliasing
- [ ] Text rendering

## Usage
See `examples/white_window.rs`
